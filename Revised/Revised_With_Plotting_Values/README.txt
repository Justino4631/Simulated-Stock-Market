This code is mostly the same as in the Revised folder, however the files have been updated to have a better user interface.
It also allows users to track stock prices and plot the changes using the python matplotlib package.

Once again you NEED to reset the values in the stock_data.xlsx file. Additionally, for better use, change the number in the Available_Money.txt file to 10,000
and in the Stocks.txt file, change all numbers to 0.
